<h2>深大寺を散歩</h2>
<p>深大寺へは調布駅から出ているバスで行きます。<br />
途中に行ってみたいそば屋さんがあったのでバスを途中下車し散歩しました。<br />
<a href="/wp-content/uploads/jindaiji_130112_01.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_01-448x300.jpg" alt="jindaiji_130112_01" width="448" height="300" class="alignnone size-large wp-image-5619" /></a><br />
深大寺は深大寺そばというそばが有名でお寺の周りにはそば屋さんが多くあります。<br />
<a href="/wp-content/uploads/jindaiji_130112_04.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_04-448x300.jpg" alt="jindaiji_130112_04" width="448" height="300" class="alignnone size-large wp-image-5622" /></a><br />
<a href="/wp-content/uploads/jindaiji_130112_06.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_06-448x300.jpg" alt="jindaiji_130112_06" width="448" height="300" class="alignnone size-large wp-image-5624" /></a><br />
▼行ったのは湧水さんというお店。<br />
<a href="/wp-content/uploads/jindaiji_130112_02.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_02-448x300.jpg" alt="jindaiji_130112_02" width="448" height="300" class="alignnone size-large wp-image-5620" /></a><br />
▼湧水さんのメニュー<br />
<a href="/wp-content/uploads/jindaiji_130112_03.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_03-448x300.jpg" alt="jindaiji_130112_03" width="448" height="300" class="alignnone size-large wp-image-5621" /></a><br />
▼山かけそばを頼みました！<br />
<a href="/wp-content/uploads/jindaiji_130112_18.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_18-448x447.jpg" alt="jindaiji_130112_18" width="448" height="447" class="alignnone size-large wp-image-5635" /></a><br />
そばを食べて腹ごしらえをして深大寺へ向かいました。<br />
▼お寺のすぐ近くにはだるま屋さんや<br />
<a href="/wp-content/uploads/jindaiji_130112_05.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_05-448x300.jpg" alt="jindaiji_130112_05" width="448" height="300" class="alignnone size-large wp-image-5623" /></a><br />
▼お茶屋さんが軒を連ねています。<br />
<a href="/wp-content/uploads/jindaiji_130112_10.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_10-448x300.jpg" alt="jindaiji_130112_10" width="448" height="300" class="alignnone size-large wp-image-5628" /></a><br />
▼美味しそうだったのでお饅頭と<br />
<a href="/wp-content/uploads/jindaiji_130112_19.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_19-448x447.jpg" alt="jindaiji_130112_19" width="448" height="447" class="alignnone size-large wp-image-5636" /></a><br />
▼甘酒で一休み。<br />
<a href="/wp-content/uploads/jindaiji_130112_20.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_20-448x447.jpg" alt="jindaiji_130112_20" width="448" height="447" class="alignnone size-large wp-image-5637" /></a><br />

▼そばが有名な深大寺には「そば守観音」が祀られています。<br />
<a href="/wp-content/uploads/jindaiji_130112_09.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_09-448x300.jpg" alt="jindaiji_130112_09" width="448" height="300" class="alignnone size-large wp-image-5627" /></a><br />
▼本殿<br />
<a href="/wp-content/uploads/jindaiji_130112_07.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_07-448x300.jpg" alt="jindaiji_130112_07" width="448" height="300" class="alignnone size-large wp-image-5625" /></a><br />

<a href="/wp-content/uploads/jindaiji_130112_08.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_08-448x300.jpg" alt="jindaiji_130112_08" width="448" height="300" class="alignnone size-large wp-image-5626" /></a><br />

▼植物の写真を撮ってみようと、初詣の帰りに神代植物公園へ行きました。<br />
<a href="/wp-content/uploads/jindaiji_130112_13.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_13-448x300.jpg" alt="jindaiji_130112_13" width="448" height="300" class="alignnone size-large wp-image-5631" /></a><br />
この時期はどんな植物があるんでしょうか。<br />
▼行ってみるとまさかの休館日！<br />
<a href="/wp-content/uploads/jindaiji_130112_21.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_21-448x448.jpg" alt="jindaiji_130112_21" width="448" height="448" class="alignnone size-large wp-image-5638" /></a><br />
ご利用は計画的に。<br />
▼行ったつもりで近くのお花屋さんの写真を1枚。<br />
<a href="/wp-content/uploads/jindaiji_130112_11.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_11-448x300.jpg" alt="jindaiji_130112_11" width="448" height="300" class="alignnone size-large wp-image-5629" /></a><br />
帰りは駅まで歩いて帰りました。<br />
<a href="/wp-content/uploads/jindaiji_130112_14.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_14-448x300.jpg" alt="jindaiji_130112_14" width="448" height="300" class="alignnone size-large wp-image-5632" /></a><br />
大通り沿いの歩道には歩行者と自転車が別れて通るような標識があちこちに目立ちました。<br />
<a href="/wp-content/uploads/jindaiji_130112_15.jpg" target="_blank"><img src="/wp-content/uploads/jindaiji_130112_15-448x300.jpg" alt="jindaiji_130112_15" width="448" height="300" class="alignnone size-large wp-image-5633" /></a><br />

<h2>最後に</h2>
<p>デジイチを持って初めての外出でした。<br />
目で見た景色とファインダー越しの景色はまた違って見えて、楽しみながらの散歩になりました。<br />
これからも色々なところへ行って写真を撮ってみたくなりました。<br />
難しいけど楽しい！<br />
▼こんなレンズで撮ってみたいです！</p>
